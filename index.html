Masukkan Personal Access Token milikmu (scope repo) untuk membuat, mengedit, dan menghapus repository. Buat token di github.com/settings/tokens.

Token tidak valid atau tidak punya akses.

Token hanya tersimpan di browser kamu sendiri (localStorage) dan dikirim langsung ke GitHub API. Tidak pernah lewat server manapun.
